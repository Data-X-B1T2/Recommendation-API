## API Reference

#### Get Recommedations

```http
  GET /recommedation/{Movie Name}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `movie_name` | `string` | **Required** Enter the name of the movie you want recommedation |
